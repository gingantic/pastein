About

Pastein is a simple pastebin service that allows you to store and share text online up to 1MB. It is a free service that allows you to create a new paste, view existing pastes, and delete pastes. You can also set a title and password for your paste.

Pastein is built using Django, a high-level Python web framework that encourages rapid development and clean, pragmatic design. The front-end is built using Bootstrap, a popular CSS framework for developing responsive and mobile-first websites.

The source code for Pastein is available on GitHub. You can view the source code, contribute to the project, and report issues on the GitHub repository.

If you have any questions, feedback, or suggestions, feel free to contact me on github or sosial below. I appreciate your feedback.

Thank you for using Pastein! We hope you find it useful for storing and sharing text online.
